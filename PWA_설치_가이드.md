# 🍲 마라탕 퀴즈 — 앱처럼 사용하기

## 1단계: 서버 시작
바탕화면(또는 폴더)의 **`마라탕퀴즈_시작.bat`** 더블클릭
→ Flask 서버 자동 시작 + 브라우저 자동 열림

## 2단계: 앱 설치 (한 번만)

### Edge
1. 주소창 우측 끝 **⊕ (앱 설치)** 아이콘 클릭
   또는 **... (메뉴) → 앱 → 이 사이트를 앱으로 설치**
2. "마라탕퀴즈" 이름 확인 → **설치**
3. ✅ 바탕화면, 시작 메뉴, 작업 표시줄 자동 등록

### Chrome
1. 주소창 우측 **⊕ 아이콘** 클릭
   또는 **⋮ (메뉴) → 마라탕퀴즈 설치**
2. 이름 확인 → **설치**
3. ✅ 바탕화면 자동 생성

## 3단계: 앱처럼 사용
- **바탕화면 마라탕 아이콘 더블클릭** → 독립 앱 창으로 실행
- 주소창 ❌ / 탭바 ❌ / 진짜 앱처럼 사용
- 전체화면 버튼 → TV 출력 시 깨끗

## ⚠️ 주의
앱이 작동하려면 **`마라탕퀴즈_시작.bat`** 으로 서버가 켜져 있어야 합니다.
(서버 없이 앱만 실행하면 데이터 로드 안 됨)

## 매번 사용 흐름
1. `마라탕퀴즈_시작.bat` 더블클릭 (서버 시작)
2. 바탕화면 마라탕 아이콘 더블클릭 (앱 실행)
3. 4팀/개인 모드 선택 → 진행
4. 끝나면 .bat 창 닫기 (서버 종료)
